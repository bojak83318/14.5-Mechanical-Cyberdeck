# Release v1.0.0: Architectural Baseline

This release marks the completion of the theoretical and architectural design phase.

## Included Specifications:
- **Mechanical:** OpenSCAD Case Design & Framework Slot Parameters.
- **Electrical:** MOSFET Low-Side Switching & I-PEX 40-pin eDP Mapping.
- **Firmware:** QMK Screen Toggle Logic.
- **Manufacturing:** ASA Smoothing & Thermal Management Protocol.

**Current Build Target:** AUO 14.5" 2560x720 Bar Display.
