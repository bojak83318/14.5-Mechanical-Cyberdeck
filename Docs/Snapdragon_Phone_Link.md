# Snapdragon + Pi 5 Dual-Brain Setup

## Hardware Connectivity
- **Primary:** Pi 5 connected to eDP Input 1.
- **Secondary:** Framework Slot (USB-C) connected to eDP Input 2.
- **Data Link:** USB-C cable between Phone and Pi 5 (using USB Ethernet/Gadget mode).

## Software Logic
- **Display Switching:** Use the 'Source' button on the AP2K1.71 or map a QMK macro to toggle inputs.
- **Data Sharing:** Enable 'USB Tethering' on the Snapdragon phone to give the Pi 5 high-speed 5G internet.
