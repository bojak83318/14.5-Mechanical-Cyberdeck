# Thermal Management Logic

- **Heat Source A:** LCD Backlight (approx. 5-7W at 900 nits).
- **Heat Source B:** AP2K1.71 Processor (approx. 2-3W during 2K scaling).
- **Cooling Strategy:** - Passive intake at front-bottom.
  - Active exhaust via 3007 (30mm) 5V Blower Fan.
  - Material Limit: ASA Glass Transition = 100°C. 
  - Expected Internal Ambient: 45°C.
