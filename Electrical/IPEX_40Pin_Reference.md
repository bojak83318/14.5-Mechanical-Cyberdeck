# I-PEX 20455-040E Pinout Reference (eDP + Touch)

| Pin | Signal | Description |
| :--- | :--- | :--- |
| 1-4 | VCC | 3.3V Power for Logic |
| 10-13 | Lane 0-1 | High-speed Video Data |
| 17-18 | AUX CH | Bidirectional Data (Touch/Handshake) |
| 20-21 | HPD | Hot Plug Detect |
| 31-36 | VLED | Backlight Power (High Voltage) |
| 38-40 | GND | Signal Ground |

**Note:** Pins 17 and 18 are the "brain" of the touch system. Any interference here will kill touch functionality.
